Bezier Curves

A Bezier curve is a parametric curve used widely in designing and computer graphics. It was first used by French engineer Pierre Bezier in the 1960s. A bezier curve is generated through the process of recursive linear interpolation of control points. All text on computers are rendered as Bezier curves, since it is more efficient to store a bunch of coordinates than to store pixel data.

Brief Description

Suppose there are N ordered control points. Then we have N-1 lines connecting them. Let 't' a fraction moved along a line ('t' is between 0 and 1). We get N-1 points by moving along 't' portion of all N-1 lines. This process is continued again, this time with N-1 ordered points and N-2 lines joining them, until we end up with just one point that constitutes the Bezier curve. The set of points obtained by varying 't' from 0 to 1 is the complete Bezier curve.
