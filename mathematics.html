Simulation of Mandelbrot Fractal

Simulation of Solution of Lorenz System
  1. Plots solution of the Lorenz system with respect to the given parameters
  2. Parameters can be altered
  3. Brief description about the Lorenz system and its importance


Visualization of Conic Sections
  1. Interactive simulation of conic sections in polar coordinate system. All parameters can be altered
  2. Circles, ellipses, parabolae and hyperbolae can be drawn
  3. Brief discussion on how they are obtained and their eccentricities


Rendering Hilbert Curves
  1. Fast and Optimised renderer of pseudo Hilbert curves
  2. Order can be changed
  3. Iterative implementation and cascade rendering implies that higher order curves can be drawn
  4. Brief description of the Hilbert curve and it's uses


Visualization of Linear Transformations
  1. Interactive simulation of a linear transformation in two dimensions
  2. The matrix representing the linear transformation can be altered. Rotation matrices are automatically entered
  3. Brief description of linear transformations, determinant and inverse of a matrix
  4. The effects of a linear transformation to a particular vector can be observed


Visualization of Trigonometric Functions
  1. Interactive presentation of all six trigonometric functions in a unit circle
  2. Existence of trigonometric functions outside of triangles and for angles more than 90 degrees.
  3. Intuition for values of trigonometric functions for multiples of 90 degrees
  4. Showcasing the odd nature of Sine and even nature of Cosine
  5. Visual proof of the three fundamental trigonometric identities
