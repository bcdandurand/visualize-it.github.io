Polynomial Regression

Polynomial regression is used to find the coefficients of an m-degree polynomial that best fits the given set of points. It involves elementary multivariable calculus and linear algebra.

Proper fitting with a polynomial of degree m requires atleast m+1 points.


Brief Description

The cost function expresses the discrepancy between the polynmial under consideration, and the given set of points. If m is the order of the polynomial being considered for a fit, then the cost function takes m+1 inputs, corresponding to the m+1 polynomial coefficients. Our aim is to find the coefficients for which the cost function is minimum or zero. Using the fact that the derivative is zero at a minima, we obtain m+1 equations by partially differentiating the cost function with respect to m+1 polynomial coefficients (the unknowns). By expressing the set of m+1 equations in terms of matrices, one can easily find the unknown coefficients using linear algebra (either Gaussian elimination + back-substitution, or inversion).

Please refer to this website for viewing the exact mathematics involved in polynomial regression.



Note:
  1. A set of n points can be exactly fitted with a polynomial of degree n-1. This is the general analogue of the fact that there is always a line (which is a 1 dimensional polynomial) joining 2 points.
  2. A polynomial of degree m requires atleast m+1 points for a proper fit. If this condition is not satisfied, then the extra coefficients will be extremely low or zero, which badly skewes the curve.
